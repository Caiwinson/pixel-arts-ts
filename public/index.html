Features

Create Canvases

Use /create canvas to start a 5×5 up to 15×15 canvas. Unlock 20×20 & 25×25 by voting.

⭐ Vote only 🎨

Custom Colors

Unlock any hex color by voting on Top.gg. Votes last 12 hours.

⭐ Vote only 🔄

Recreate Images

Transform any image into an editable pixel canvas with /recreate.

📼

Timelapse

Generate a speed-adjustable timelapse video of your canvas evolution.

How to Use
1

Run /create canvas to generate a blank canvas.

2

Click the grid buttons to fill pixels with your selected color.

3

Use the color picker or unlock custom colors by voting.

4

Use advanced tools — line, fill, replace color — for complex designs.

5

Use /recreate to turn any image into an editable canvas.
